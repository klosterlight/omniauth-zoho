# Zoho::Omniauth
Strategy to authenticate with Zoho via OAuth2 in OmniAuth.

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'omniauth-zoho'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install omniauth-zoho
```
